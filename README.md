# 🚀 PrepWise AI

PrepWise AI is a full-stack mock interview platform built to help students practice technical interviews in a more interactive and structured way.

The application includes secure authentication, protected routes, dashboard functionality, responsive UI, and complete frontend-backend integration with production deployment.

This project was built to strengthen practical full stack development skills and gain hands-on experience with authentication systems, API integration, deployment workflows, and real-world debugging.

# ⚙️ Installation & Setup

## Clone Repository

```bash
git clone https://github.com/kl2400032639/prepwise-ai.git
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## Backend Setup

```bash
cd backend
npm install
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000
JWT_SECRET=your_secret_key
```
